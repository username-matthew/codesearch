## LAMP Stack CRUD with Authentication/Authorisation and Search

#### Stack
- PHP
- MYSQL
- JS

### Functionality

- Register
- Login
- Create
- Read
- Update/Edit
- Delete
- Search

`Add your database configuration to a file called dbconfig.php and save it in the db directory`
